# 技能-n8n工作流

## 目的

為本專案維護 n8n workflow 規格與可匯入 JSON。

## 第一批 workflow

1. `stats`
2. `list`
3. `query`
4. `dashboard`
5. `submit`
6. `inspections`
7. `interpellations`
8. `proposals`
9. `achievements`
10. `achievements/like`
11. `pages`
12. `pages/:slug`

## 規則

1. webhook path 統一由客戶 prefix 管理
2. Notion 欄位 mapping 要固定
3. response body 保持一致格式
4. JSON 檔命名要能直接辨識用途
